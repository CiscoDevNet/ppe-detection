ppe-backend
================

## Installation
1. Install Dependencies

	```
	$ pip install -r requirements.txt
	```


2. Run dependencies services

	```
	$ docker-compose up -d
	```

3. Start the server

	```
	$ python main.py
	```

Your server should now be running on [0.0.0.0:7030](http://0.0.0.0:7030/).
